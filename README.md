# Анализ популярности треков Spotify

## Описание проекта
Исследование факторов, влияющих на популярность треков на Spotify. 
В проекте использованы данные с Kaggle, обогащенные через MusicBrainz API.

## Используемые технологии
- Python, pandas, numpy
- Scikit-learn (Linear, Ridge, Lasso, SVR)
- SHAP для интерпретации моделей
- MusicBrainz API для обогащения данных

## Результаты
- Лучшая модель: SVR (R² ~ 0.20-0.28)
- Ключевые факторы популярности:
  - Активность исполнителя
  - Страна исполнителя (US, PR, KR)
  - Жанр (Hip-Hop/R&B/Soul)

## Как запустить
1. Установите зависимости: `pip install -r requirements.txt`
2. Откройте ноутбук: `jupyter notebook АДМО_Гареева_лр1.ipynb`
3. Запустите все ячейки
